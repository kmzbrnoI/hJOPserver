# hJOPserver

Main technological server of [hJOP project](https://hjop.kmz-brno.cz/).

* Programming language: Object Pascal
* Environment: Delphi 2009
* Author: Jan Horacek
* For: Microsoft Windows OS

## Notes

* Do not forget to load submodules.
